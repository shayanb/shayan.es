## Development

### Local Setup

```bash
# Clone the repository
git clone https://github.com/shayanb/shayan.es.git
cd shayan.es

# Install dependencies
bundle install

# Run development server
bundle exec jekyll serve

# Build for production
bundle exec jekyll build
```
